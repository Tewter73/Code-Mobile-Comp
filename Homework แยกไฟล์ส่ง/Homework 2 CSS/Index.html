คาร์ดิโอ (Cardio Training)

คาร์ดิโอ (Cardiovascular Exercise) คือการออกกำลังกายที่เน้นการกระตุ้นระบบไหลเวียนโลหิต อัตราการเต้นของหัวใจ และปอดให้ทำงานอย่างต่อเนื่อง ช่วยเผาผลาญไขมันสะสมและเพิ่มความอึดให้ร่างกาย


รูปแบบของการทำคาร์ดิโอ:

- LISS (Low-Intensity Steady State): คาร์ดิโอความเข้มข้นต่ำแต่ต่อเนื่อง

- MICT (Moderate-Intensity Continuous Training): คาร์ดิโอระดับปานกลาง

- HIIT (High-Intensity Interval Training): คาร์ดิโอความเข้มข้นสูงสลับพัก


อุปกรณ์หรือกิจกรรมยอดนิยม: ลู่วิ่งไฟฟ้า, เครื่องเดินวงรี, กระโดดเชือก, ว่ายน้ำ, ปั่นจักรยาน
